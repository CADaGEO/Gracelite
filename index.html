Gracelite

Gracelite est un projet de toolbox dédié à l'exploitation du Geostandard ANT (Aménagement Numérique des Territoires) avec Spatialite.
Geostandard ANT.
Le nom d'usage du projet à la base du Geostandard ANT est GraceTHD. Historiquement Gr@ce est le nom du portail de géoréférencement et de recensement des infrastructures de communications électroniques de la région Aquitaine. http://grace.aquitaine.fr/accueil/ Le modèle de données de ce portail a massivement servi de modèle au Geostandard ANT.

Une version 2 du Geostandard ANT est en cours de préparation. Gracelite vise cette version. Le modèle de données de données est géré sur ce dépôt : GraceTHD-MCD.

Utilisation

Un lanceur est disponible pour débuter avec Gracelite : user_launch.bat
